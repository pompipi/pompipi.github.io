How to install ArchLinux on WSL2 from scratch

How to install ArchLinux on WSL2 from scratch 19 Feb 2022 Install Docker desktop on windows and enable WSL2 integration Need an existing WSL2 distribution already installed, because we need to run Docker from there. Use Ubuntu. Install ArchLinux Docker container from Ubuntu WSL2. This will be the basis of the distribution installed on WSL2. Run the container, and setup some things like users etc. Add user to wheel group, and enable by visudo and uncomment: ## Uncomment to allow members of group wheel to execute any command with no passwd %wheel ALL=(ALL) ALL NOPASSWD: ALL Export image into tar file Register ArchLinux tar file to WSL2 To set default login user, edit wsl.
